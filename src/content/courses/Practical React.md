# Aim
The aim of this course is to equip you with the tools and abilities of a react wielding web developer, you will be able to start producing react websites after this, as well as understand where and how to learn more depending on what you want to specialise in.

The purpose of this course to give a no-bullshit course of the fundamentals of react website development. 
Ai is not demonised in this course, but rather along with other tools, you will learn to use it with precision rather than abusing it and add to [vibe coding](https://en.wikipedia.org/wiki/Vibe_coding#:~:text=In%20computer%20programming%2C%20vibe%20coding,vulnerabilities%20in%20the%20resulting%20software)
